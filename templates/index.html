Sobre o Sistema de Monitoramento
Sistema de Monitoramento da Qualidade do Ar em Tempo Real

Este sistema foi desenvolvido como trabalho prático para a disciplina de Big Data do curso da UFRJ/CEDERJ, sob orientação do Prof. Alexandre de Assis Bento Lima.

Utiliza tecnologias de ponta para processamento de dados em tempo real, demonstrando a característica de Velocidade do Big Data através do Apache Spark Structured Streaming.

Tecnologias Utilizadas
  • Apache Spark (PySpark) para processamento em tempo real
  • Python/Flask para o backend
  • HTML/CSS/JavaScript para o frontend
  • Bootstrap para design responsivo
